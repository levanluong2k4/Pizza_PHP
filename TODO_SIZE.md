# TODO: Quản lý size sản phẩm và giá theo size cho admin

## Quản lý size sản phẩm (2 trang)
### 1. Danh sách size sản phẩm
- Tạo admin/view/list_sizes.php với bảng hiển thị tất cả sizes
- Chức năng sửa và xóa size

### 2. Thêm size sản phẩm
- Tạo admin/view/add_size.php với form thêm size mới

## Quản lý giá theo size (2 trang)
### 3. Danh sách giá sản phẩm theo size
- Cải thiện admin/view/view_sanphamsize.php thành admin/view/list_product_prices.php
- Hiển thị sản phẩm, size, giá, ảnh với Bootstrap

### 4. Thêm giá sản phẩm
- Cải thiện admin/view/insert_product_size.php thành admin/view/add_product_price.php
- Form Bootstrap với select sản phẩm, size, giá, upload ảnh

## File xử lý cần tạo
- admin/process/add_size.php
- admin/process/edit_size.php
- admin/process/delete_size.php
